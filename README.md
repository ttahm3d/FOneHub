# Features

- User
  - Create account
  - Edit your profile
  - Follow / unfollow users
- Posts
  - Create Posts
  - Edit posts
  - Delete Posts
  - Add Comments to posts
  - Like posts
  - Bookmark posts

# Technologies Used

- React
- Redux - Redux Toolkit
- Firebase
  - Google Auth
- Styled Components
